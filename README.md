# theutia

A TypeScript Web Application Framework

# for Developer

## Requirements

- Node.js LTS
- yarn

## Bootstrapping

```bash
git clone git@github.com:ndxbn-tokyo/theutia.git
cd theutia

git config commit.template .commit-message-template.txt
yarn
```
